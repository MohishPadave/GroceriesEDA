# Groceries Notebook

This repository contains a notebook named `groceries.ipynb`.

The notebook works with a groceries dataset and includes data loading, basic cleaning, and a few visualizations to understand the data. You can open it directly on GitHub or run it locally.

### Requirements
- Python 3
- Jupyter Notebook
- pandas
- numpy
- matplotlib

### How to run
1. Open the notebook
2. Run the cells in order

Feel free to explore or build on top of it.
